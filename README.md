Brauzerda ishlaydigan, responsiv, dark/light rejimli oddiy kalkulyator.  
HTML + CSS + Vanilla JavaScript bilan yozilgan – hech qanday framework ishlatilmagan.

https://your-username.github.io/simplecalc/   ← Live Demo (keyin o‘zingizniki bilan almashtiring)

## Xususiyatlar

- Asosiy matematik amallar: + − × ÷
- Belgini o‘zgartirish (+/-)
- Foiz hisoblash (%)
- Oxirgi belgini o‘chirish (←)
- To‘liq tozalash (AC)
- Klaviatura orqali to‘liq qo‘llab-quvvatlash (raqamlar, operatorlar, Enter, Backspace, Escape)
- Chain calculation (ketma-ket hisoblash: 5 + 3 × 2 = ...)
- Xatolarni aniqlash (masalan: 0 ga bo‘lish)
- Oxirgi 5 ta hisoblash tarixi (history)
- Natijani clipboardga nusxalash
- Dark / Light mode (avtomatik yoki qo‘lda tanlash)
- To‘liq responsiv dizayn (telefon, planshet, desktop)

## Texnologiyalar

- HTML5  
- CSS3 (Flexbox + Grid)  
- Vanilla JavaScript (ES6+)  
- Font: system-ui yoki Inter (Google Fonts)  
- Hosting: GitHub Pages / Vercel / Netlify
